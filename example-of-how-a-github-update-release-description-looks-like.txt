## Studio Update

Download `CrosshairY.exe` below and run it. No installer needed. Requires the [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) if not already installed.

### What's new in v1.0.5

- **Builder tools** - the crosshair builder now has a full tool set: pencil, eraser, fill (bucket), line, rectangle and ellipse, with a live preview as you drag out shapes
- **Undo / redo** - step back and forward through your edits with the buttons or Ctrl+Z / Ctrl+Y
- **Mirror symmetry** - draw with left/right, up/down or 4-way mirroring so symmetric crosshairs come together in seconds
- **Save as crosshair** - pin a drawing from the builder into the new "My Crosshairs" grid and switch to it in one click, no rebuilding
- **Sharper drawn crosshairs** - custom crosshairs now render as clean, crisp vectors on the overlay instead of looking blocky
- **5 new templates** - Dot Ring, 2 Rings, Plus·, Corners and X Thick, bringing the built-in count to 22
- **Multi-monitor** - choose which display the crosshair appears on under Settings
- **More private survey backend** - survey responses now route through a server instead of shipping any webhook inside the app
- **Various bug fixes and polish**
